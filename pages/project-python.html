Optimizando procesos académicos mediante Python y Selenium.

Sobre el Proyecto

Este proyecto nació de la necesidad de gestionar eficientemente las actividades del campus virtual. Utilizando Python y librerías de automatización de alto nivel, el sistema es capaz de:

  • Autenticación automática en portales educativos.
  • Extracción de fechas límite de tareas y exámenes.
  • Sincronización con bases de datos locales y calendarios.
  • Generación de reportes automáticos en PDF o Excel. 
import selenium
from bs4 import BeautifulSoup

def scrape_campus():
    driver = webdriver.Chrome()
    driver.get(CAMPUS_URL)
    # Lógica de login...
    soup = BeautifulSoup(driver.page_source)
    tasks = soup.find_all('div', class_='task')
    return tasks
