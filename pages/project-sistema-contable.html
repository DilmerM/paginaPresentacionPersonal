Sobre el Proyecto

ContaHN es una plataforma de contabilidad de próxima generación diseñada específicamente para el mercado hondureño. Combina la potencia de C# y .NET con la agilidad de React para ofrecer un sistema robusto, escalable y fácil de usar.

El sistema integra análisis estadístico avanzado utilizando R, permitiendo a las empresas no solo llevar su contabilidad, sino también obtener insights estratégicos sobre sus gastos, ingresos y salud financiera general.

Con un fuerte enfoque en el cumplimiento de las normativas de la SAR 2024, ContaHN garantiza que las empresas operen dentro del marco legal mientras optimizan sus procesos internos de nómina, inventario y facturación.

Stack Tecnológico

  • C#
  • .NET
  • React
  • R (Análisis de Datos)
